Toon — peau & tenue
Même squelette, mêmes animations, deux modules de rendu. À gauche le rendu stick (v0.2), à droite le rendu toon : peau, coiffure, t-shirt, pantalon, mains, chaussures — tout est encore généré par le code (aucun asset dessiné), donc personnalisation illimitée : peaux, tenues et coiffures sont de simples paramètres du JSON personnage (style.rendu = "stick" | "toon"). Les 17 actions, les émotions et le lip sync fonctionnent à l'identique sur les deux.
